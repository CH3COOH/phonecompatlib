PhoneCompatibilityToolkit
==========
ストアアプリでPhoneアプリの資産を生かしたくて生かしたくてたまらない、互換性がないなんてやだあああああ！そんな貴方のためのネタライブラリです。

なにができるの
----------
ストアアプリでPhoneアプリのこんな機能が使えるようになるかも知れません。
* ApplicationBar
* ContextMenu
* IsolatedStorageSettings
* Location
* MessageBox
* NavigationService
* XxxTask
ときたま未実装もあります。

結局なんなの
----------
ネタライブラリです。真に受けないでください。電話には電話の。パソコンにはパソコンのUXがあります。ちゃんとそこらへんを理解してアプリ作りましょうね！
